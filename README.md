# CoDeSys_Pump_Control
Practical work on creating a system for control and visualize state of pump
Проект управления насосом на виртуальном контроллере CoDeSys Win. Алгоритм написан на языке ST, создана визуальная панель отображающая состояние насоса и элементы управления насосом.

Цель:
- создать функциональный блок насоса;
- сделать визуализацию для просмотра состояния и упраления насосом.

Описание созданной системы:

2. Логика работы системы управления насосом:
  - ручной пуск;
  - ручной останов;
  - отсанов по сигналу от датчика сухого хода с включением задержки на ручной пуск 15 сек.;
  - останов по сигналу от датчика перегруза насососа (датчик температуры);
  - выдача аварии и выдача сигнала задержки пуска.
<img width="1236" height="1031" alt="image" src="https://github.com/user-attachments/assets/b77ae52a-90f9-4335-bb78-760e2528befd" />

3. Визуализация
  - отображение индикаторов состояния насоса: включен/выключен;
  - кнопки ручного пуска и останова;
  - индикатор и таймер задержки ручного пуска после сработ ки датчика сухого хода;
  - индикатор Аварии в системе;
  - кнопка ручного сброса Аварии.

Результат:
Создан алгоритм работы системы управления насосом на основе функционального блока. Данную систему можно легко масштабировать, увеличивая количество насосов, просто добавляя новый объект на основе ФБ и входные/выходные переменые для него.
